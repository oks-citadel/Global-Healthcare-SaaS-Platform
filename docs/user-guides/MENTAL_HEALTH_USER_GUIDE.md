### Taking Self-Assessments

#### Available Assessments

| Assessment | Purpose                | Time    |
| ---------- | ---------------------- | ------- |
| **PHQ-9**  | Depression screening   | 3-5 min |
| **GAD-7**  | Anxiety screening      | 2-3 min |
| **PHQ-2**  | Quick depression check | 1 min   |
| **PCL-5**  | PTSD screening         | 5-7 min |
| **AUDIT**  | Alcohol use assessment | 3-5 min |

#### Step 1: Select an Assessment

1. Go to **"Mental Health"** > **"Assessments"**
2. Choose the assessment to complete
3. Read the instructions carefully

#### Step 2: Complete the Questions

1. Answer each question honestly
2. Select the response that best describes how you've felt
3. Consider the timeframe specified (usually past 2 weeks)

#### Example: PHQ-9 Questions

```
Over the last 2 weeks, how often have you been bothered by:

1. Little interest or pleasure in doing things?
   ○ Not at all (0)
   ○ Several days (1)
   ○ More than half the days (2)
   ○ Nearly every day (3)

2. Feeling down, depressed, or hopeless?
   ○ Not at all (0)
   ○ Several days (1)
   ○ More than half the days (2)
   ○ Nearly every day (3)

[Continue for all 9 questions...]
```

#### Step 3: View Your Results

After completing the assessment:

1. View your score and severity level
2. Read interpretation guidelines
3. See recommended next steps
4. Results are shared with your care team

#### Understanding PHQ-9 Scores

| Score | Severity          | Recommendation          |
| ----- | ----------------- | ----------------------- |
| 0-4   | Minimal           | Self-care resources     |
| 5-9   | Mild              | Consider counseling     |
| 10-14 | Moderate          | Recommend treatment     |
| 15-19 | Moderately Severe | Active treatment needed |
| 20-27 | Severe            | Immediate consultation  |

### Scheduling Therapy Sessions

#### Step 1: Request a Session

1. Go to **"Mental Health"** > **"Schedule Session"**
2. Select session type:
   - **Individual Therapy**: One-on-one with therapist
   - **Group Therapy**: Group sessions on specific topics
   - **Couples/Family**: Sessions with partners or family

#### Step 2: Choose Your Therapist

1. Browse available therapists
2. View profiles including:
   - Credentials and specialties
   - Treatment approaches (CBT, DBT, etc.)
   - Languages spoken
   - Availability
3. Select your preferred therapist

#### Step 3: Select Date and Time

1. View available time slots
2. Choose in-person or video session
3. Select appointment duration (typically 45-60 minutes)
4. Confirm your selection

#### Step 4: Complete Pre-Session Forms

Before your first session:

1. Complete intake questionnaire
2. Review confidentiality policies
3. Sign consent forms electronically
4. Provide emergency contact information

### Crisis Resources

#### If You're in Crisis

**If you or someone you know is in immediate danger, call 911.**

#### Crisis Hotlines

| Resource                                 | Contact             |
| ---------------------------------------- | ------------------- |
| **National Suicide Prevention Lifeline** | 988                 |
| **Crisis Text Line**                     | Text HOME to 741741 |
| **SAMHSA National Helpline**             | 1-800-662-4357      |
| **Veterans Crisis Line**                 | 988, Press 1        |

#### Using the Crisis Support Feature

1. Click **"Crisis Support"** button (available 24/7)
2. Immediately see:
   - Emergency hotline numbers
   - Text-based crisis support
   - Safety resources
3. Option to alert your care team

#### Safety Plan

If you have a safety plan:

1. Access it anytime at **"Mental Health"** > **"My Safety Plan"**
2. View your:
   - Warning signs
   - Coping strategies
   - People to contact
   - Safe places to go
   - Reasons for living

## For Clinicians

### Conducting Assessments

#### Administering Assessments

1. From patient chart, go to **"Mental Health"** > **"Assessments"**
2. Click **"New Assessment"**
3. Select assessment type:
   - PHQ-9 (Depression)
   - GAD-7 (Anxiety)
   - C-SSRS (Suicide Risk)
   - PCL-5 (PTSD)
   - AUDIT (Alcohol Use)

#### In-Session Assessment

1. Select **"Clinician-Administered"**
2. Ask questions verbally and record responses
3. System auto-calculates score
4. Review results with patient

#### Patient Self-Assessment

1. Select **"Patient Self-Report"**
2. Send assessment link to patient
3. Patient completes on their device
4. Results appear in chart when complete

### Creating Treatment Plans

#### Step 1: Initiate Treatment Plan

1. Go to patient chart > **"Mental Health"** > **"Treatment Plans"**
2. Click **"New Treatment Plan"**
3. Enter primary diagnosis (ICD-10)

#### Step 2: Set Treatment Goals

Add specific, measurable goals:

```
┌─────────────────────────────────────────────────────────────────┐
│  ADD TREATMENT GOAL                                              │
├─────────────────────────────────────────────────────────────────┤
│  Goal Description:                                              │
│  [Reduce depressive symptoms as measured by PHQ-9]              │
│                                                                 │
│  Target Measure:                                                │
│  Baseline: [PHQ-9 = 17]     Target: [PHQ-9 ≤ 9]                │
│                                                                 │
│  Target Date: [04/15/2025] (3 months)                          │
│                                                                 │
│  Strategies/Interventions:                                      │
│  ☑ Cognitive Behavioral Therapy (weekly sessions)              │
│  ☑ Behavioral activation homework                              │
│  ☑ Sleep hygiene education                                     │
│  ☑ Consider medication if no improvement by week 4             │
│                                                                 │
│  [Add Goal] [Cancel]                                            │
└─────────────────────────────────────────────────────────────────┘
```

#### Step 3: Define Interventions

| Intervention Type   | Examples                      |
| ------------------- | ----------------------------- |
| **Psychotherapy**   | CBT, DBT, EMDR, IPT           |
| **Skills Training** | Coping skills, mindfulness    |
| **Medication**      | Antidepressants, anxiolytics  |
| **Lifestyle**       | Exercise, sleep hygiene       |
| **Support**         | Group therapy, support groups |

#### Step 4: Set Review Schedule

1. Define review frequency (weekly, monthly)
2. Schedule reassessment dates
3. Set plan expiration/renewal date
4. Patient signs treatment plan

### Crisis Intervention Protocol

#### Identifying Crisis

**High-Risk Indicators:**

- PHQ-9 Item 9 score ≥ 2
- C-SSRS positive for suicidal ideation
- Patient expresses thoughts of self-harm
- Acute psychotic symptoms
- Intoxication with suicidal statements

#### Step 1: Conduct Risk Assessment

1. System auto-triggers for high PHQ-9 scores
2. Complete **Columbia Suicide Severity Rating Scale (C-SSRS)**
3. Document:
   - Suicidal ideation (passive vs active)
   - Intent
   - Plan
   - Means
   - Timeline

#### Step 2: Determine Risk Level

| Risk Level   | Indicators                            | Action                                           |
| ------------ | ------------------------------------- | ------------------------------------------------ |
| **Low**      | Passive ideation, no plan             | Safety plan, close follow-up                     |
| **Moderate** | Active ideation, vague plan           | Safety plan, frequent contact, consider referral |
| **High**     | Active ideation, specific plan, means | Immediate intervention, possible hospitalization |
| **Imminent** | Intent to act now                     | Do not leave alone, emergency services           |

#### Step 3: Implement Safety Measures

1. **Create/Review Safety Plan**
   - Warning signs
   - Coping strategies
   - Reasons for living
   - Support contacts
   - Professional contacts
   - Means restriction

2. **Document Everything**
   - Use crisis documentation template
   - Record all interventions
   - Note patient's response

3. **Follow-Up**
   - Schedule within 24-48 hours
   - Provide crisis hotline numbers
   - Notify emergency contacts (with consent)

### Medication Management

#### Prescribing Psychotropic Medications

1. Go to patient chart > **"Mental Health"** > **"Medications"**
2. Click **"Prescribe Medication"**
3. Select medication class:
   - Antidepressants (SSRIs, SNRIs, etc.)
   - Anxiolytics
   - Mood Stabilizers
   - Antipsychotics
   - Stimulants

#### Monitoring Requirements

| Medication Class | Monitoring                         |
| ---------------- | ---------------------------------- |
| SSRIs/SNRIs      | Suicidality in first weeks         |
| Lithium          | Lithium levels, renal, thyroid     |
| Antipsychotics   | Metabolic panel, weight, EPS       |
| Stimulants       | Blood pressure, heart rate, weight |
| Benzodiazepines  | Dependence, PDMP check             |

## Privacy and Confidentiality

### 42 CFR Part 2 Protections

Mental health and substance use records have additional protections:

- Written consent required for any disclosure
- Consent must specify:
  - Who can receive information
  - What information can be shared
  - Purpose of disclosure
  - Expiration date
- Records cannot be re-disclosed without new consent

### Managing Consent

#### For Patients

1. Go to **"Mental Health"** > **"Privacy Settings"**
2. View/manage consent authorizations
3. Revoke consent at any time

#### For Clinicians

1. Verify consent before sharing information
2. Document consent in chart
3. Use consent management module

## Troubleshooting

### Common Issues

#### "Cannot Access Mental Health Section"

- Verify you have mental health services enabled
- Contact your primary care provider for referral
- Check with support if technical issue

#### "Assessment Won't Submit"

- Ensure all questions are answered
- Check internet connection
- Try refreshing the page
- Use a different browser

#### "Cannot See My Therapist's Availability"

- Therapist may have limited hours
- Check if accepting new patients
- Try different dates/times

### Getting Help

| Issue              | Contact                   |
| ------------------ | ------------------------- |
| Technical problems | support@unifiedhealth.io  |
| Clinical questions | Your treatment provider   |
| Crisis             | 988 or emergency services |
| Privacy concerns   | privacy@unifiedhealth.io  |
