### Requesting a Refill

#### Step 1: Select Medication to Refill

1. Go to **"Pharmacy"** > **"Active Medications"**
2. Find the medication you need refilled
3. Click the **"Refill"** button

#### Step 2: Choose Refill Options

1. **Pharmacy Selection**:
   - Use your default pharmacy
   - Or select a different pharmacy

2. **Pickup or Delivery**:
   - **Pickup**: Ready for collection at pharmacy
   - **Delivery**: Ship to your address (if available)

3. **Quantity**:
   - Standard refill (30, 60, or 90 days)
   - Request different quantity if needed

#### Step 3: Submit Request

1. Review refill details
2. Click **"Submit Refill Request"**
3. Receive confirmation number

#### Step 4: Track Your Refill

| Status               | Meaning                       |
| -------------------- | ----------------------------- |
| **Submitted**        | Request sent to pharmacy      |
| **Processing**       | Pharmacy preparing medication |
| **Ready for Pickup** | Available at pharmacy         |
| **Out for Delivery** | Being shipped to you          |
| **Completed**        | Picked up or delivered        |

### Transferring a Prescription

#### When to Transfer

- Moving to a new area
- Prefer a different pharmacy
- Need better pricing
- Pharmacy hours don't work for you

#### Step 1: Initiate Transfer

1. Go to **"Pharmacy"** > **"My Prescriptions"**
2. Select the prescription to transfer
3. Click **"Transfer to Another Pharmacy"**

#### Step 2: Select New Pharmacy

1. Search for pharmacies by:
   - Location (address or zip code)
   - Name (CVS, Walgreens, etc.)
   - Distance from you
2. View pharmacy details:
   - Hours of operation
   - Services offered
   - Contact information
3. Select your preferred pharmacy

#### Step 3: Confirm Transfer

1. Review transfer details
2. Acknowledge that:
   - Transfer may take 24-48 hours
   - Original pharmacy will be notified
3. Click **"Confirm Transfer"**

- Phone number
- Hours of operation
- Services available
- Patient ratings

#### Step 4: Set as Default

1. Click **"Set as My Pharmacy"** to make it your default
2. Future prescriptions will be sent here

### Managing Prescription Requests

#### Viewing Refill Requests

1. Go to **"Orders"** > **"Prescription Requests"**
2. Filter by:
   - Status (pending, approved, denied)
   - Date range
   - Patient name
   - Medication type

#### Approving a Refill

1. Click on the refill request
2. Review:
   - Last fill date
   - Adherence history
   - Recent visits
   - Current conditions
3. Click **"Approve"** to authorize refill

#### Denying a Refill

1. Click on the refill request
2. Select denial reason:
   - Need appointment first
   - Medication change required
   - Safety concern
   - No longer prescribed
3. Add message to patient
4. Click **"Deny with Message"**

### Reviewing Patient Medications

#### Medication Reconciliation

1. Open patient chart
2. Go to **"Medications"** tab
3. View:
   - Active prescriptions from your practice
   - Prescriptions from other providers
   - OTC medications (patient-reported)
   - Supplements

#### Actions Available

| Action          | Use Case                                |
| --------------- | --------------------------------------- |
| **Renew**       | Continue same medication                |
| **Modify**      | Change dose, frequency, or instructions |
| **Discontinue** | Stop medication                         |
| **Substitute**  | Switch to different medication          |

### Handling Controlled Substances

#### EPCS Requirements

For Schedule II-V medications:

1. **Two-Factor Authentication** required
2. **Identity Proofing** must be current
3. **DEA Registration** verified in system

#### Step 1: Select Controlled Substance

1. Search and select medication
2. System verifies DEA schedule

#### Step 2: Complete PDMP Check

1. System automatically queries Prescription Drug Monitoring Program
2. Review patient's controlled substance history
3. Document review in chart

#### Step 3: Two-Factor Authentication

1. Enter prescription details
2. Click **"Sign with 2FA"**
3. Enter authentication code from:
   - Authenticator app
   - Hardware token
   - SMS code
4. Prescription is signed and sent

## For Pharmacists

### Processing Prescriptions

#### Receiving Prescriptions

1. Log in to **Pharmacy Portal**
2. Go to **"Incoming Prescriptions"**
3. View new e-prescriptions in queue

#### Verification Steps

1. **Patient Verification**
   - Confirm patient identity
   - Verify date of birth
   - Check insurance eligibility

2. **Prescription Validation**
   - Verify prescriber credentials
   - Check DEA number for controlled substances
   - Review for completeness

3. **Clinical Review**
   - Run drug utilization review (DUR)
   - Check for interactions with patient's profile
   - Verify appropriate dosing

#### Dispensing

1. Fill prescription
2. Apply labels
3. Counseling points documented
4. Mark as **"Ready for Pickup"**

---

### Managing Inventory

#### Checking Stock

1. Go to **"Inventory"** > **"Stock Levels"**
2. Search by medication name or NDC
3. View current quantity and reorder point

#### Low Stock Alerts

- Automated alerts when below threshold
- Suggested order quantities
- Alternative medications if out of stock

## Drug Interaction Alerts

### Alert Severity Levels

| Level        | Color  | Action                          |
| ------------ | ------ | ------------------------------- |
| **Critical** | Red    | Must address before prescribing |
| **Major**    | Orange | Requires clinical review        |
| **Moderate** | Yellow | Monitor patient closely         |
| **Minor**    | Blue   | Informational only              |

### Common Interactions

| Drug 1         | Drug 2       | Interaction             |
| -------------- | ------------ | ----------------------- |
| Warfarin       | NSAIDs       | Increased bleeding risk |
| Metformin      | Contrast dye | Lactic acidosis risk    |
| SSRIs          | MAOIs        | Serotonin syndrome      |
| ACE Inhibitors | Potassium    | Hyperkalemia risk       |

## Troubleshooting

### Common Issues

#### "Prescription Not Received at Pharmacy"

1. Verify correct pharmacy selected
2. Check prescription status in portal
3. Call pharmacy to confirm receipt
4. Contact support if not resolved

#### "Refill Too Early"

- Insurance may not cover early refills
- Check "Earliest Fill Date" in prescription details
- Request vacation override if traveling

#### "Prior Authorization Required"

1. You will receive notification
2. Provider submits PA request
3. Insurance reviews (usually 24-72 hours)
4. Track status in portal

#### "Medication Out of Stock"

1. Pharmacy will contact you
2. Options:
   - Wait for restock
   - Transfer to another pharmacy
   - Request alternative medication
